# Password-Generator
Application that generates a random password based on user-selected criteria.


### Links
* [Password-Generator Website](https://ooledezma.github.io/Password-Generator/)

### Summary
* Created an application that generates a random password based on user-selected criteria. This app will run in the browser and feature dynamically updated HTML and CSS powered by your JavaScript code. It will also feature a clean and polished user interface and be responsive, ensuring that it adapts to multiple screen sizes.

### How to Use
* The User must click the button to generate a password to start.
* The user will be asked to enter a password lenght between 8 and 128 characters.
* The user will be prompted to choose from the following password criteria: 
    * Include special characters
    * Include numbers
    * Include uppercase
    * Include lowercase 
* The password will be displayed in the page.
